# 个人博客

使用 [Docusaurus 2](https://v2.docusaurus.io/)静态网站生成器构建。

### 安装

```
$ yarn
```

### 本地开发

```
$ yarn start
```

### Build

```
$ yarn build
```

### 部署

```
$ GIT_USER=<Your GitHub username> USE_SSH=true yarn deploy
```
